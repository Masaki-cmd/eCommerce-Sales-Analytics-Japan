# eCommerce-Sales-Analytics-Japan
# E-commerceデータ分析プロジェクト：売上・利益最大化のための戦略提言

## 1. プロジェクト概要

本プロジェクトは、提供されたE-commerceのトランザクションデータ (`data.csv`) を用いて、企業の売上および利益を最大化するためのデータ駆動型戦略を策定することを目的としています。

「課題定義 → 仮説 → 分析 → 提案」のフレームワークに基づき、実務に即したデータ加工、論理的な分析プロセス、そして現場で実行可能な具体的なアクションプランを提案します。

### 目的

*   売上トレンド、商品収益性、顧客行動の深掘り。
*   売上・利益向上に繋がる具体的な課題の特定。
*   データに基づいた実行可能な改善策の提案。

### 使用データ

*   `archive.zip`に含まれる`data.csv` (E-commerceトランザクションデータ)

### フレームワーク

1.  **課題定義**: データから現状のビジネス課題を明確化。
2.  **仮説**: 課題解決に向けた仮説を立案。
3.  **分析**: 仮説検証のためのデータ分析と可視化。
4.  **提案**: 分析結果に基づく具体的なアクションと、ビジネスインパクトの提示。

## 2. データ準備とクリーニング

実務においてデータの「泥臭さ」と向き合うプロセスを重視し、以下のステップでデータを整形しました。

### 2.1. データ読み込みと初期確認

`archive.zip`を解凍し、`data.csv`を`encoding='latin1'`で読み込みました。初期確認により、以下の課題が特定されました。

*   `InvoiceDate`: `object`型であり、日付としての分析には不適切。
*   `Description`: 約0.2%の欠損値が存在。
*   `CustomerID`: 約25%の欠損値が存在。
*   `Quantity` & `UnitPrice`: 負の値が存在し、返品やデータエラーを示唆。

### 2.2. データ型の変換

`InvoiceDate`列を`datetime`型に変換し、時間ベースの分析を可能にしました。

### 2.3. 欠損値の処理

*   **`Description`**: 欠損値を持つ1,454行を削除しました。これにより、商品特定と分析の整合性を保ちました。
*   **`CustomerID`**: 顧客IDが欠損している行は、顧客ごとの分析（例: RFM分析）からは除外する方針としました。これは、特定の顧客を追跡できないためですが、全体売上や商品トレンド分析では活用します。

### 2.4. 不正な数量（Quantity）と単価（UnitPrice）の処理

*   **負の`Quantity`**: これらは返品注文として扱います。`InvoiceNo` が 'C' で始まるものもキャンセル注文として区別し、これらを通常の販売データから分離します。返品データはコスト分析や顧客満足度分析に重要ですが、売上分析では除外します。
*   **負の`UnitPrice`**: これは不正なデータである可能性が高いです。物理的に単価がマイナスになることはないため、これらの行は削除します。もしこれらが割引等を表す場合は、別途割引額として処理すべきですが、現在のデータ構造からは判断が難しいため、一旦削除します。

この結果、元データ541,909行から、`Description`の欠損値と不正な`Quantity`/`UnitPrice`の行を削除し、最終的に530,104行のクリーンなデータセットで分析を進めます。

### 2.5. 売上額（Sales）列の作成

`Quantity`と`UnitPrice`を乗算し、`Sales`列（売上額）を作成しました。これは「収益の最大化」という最終KGIを評価するための主要な指標となります。

## 3. データ分析結果：収益最大化に向けた課題特定

データクリーニング後、以下の3つの主要な観点から分析を行い、売上・利益最大化に向けた課題と機会を特定しました。

### 3.1. 全体的な売上トレンドの分析

**分析目的**: ビジネス全体の売上推移を把握し、季節性や成長傾向を理解する。

**結果**: 
*   2010年12月から2011年11月にかけて、月次売上は増加傾向を示しています。
*   特に**11月が売上のピーク**であり、年末商戦の重要性が示唆されます。
*   **2月と4月に売上が落ち込む季節性**が確認されました。
*   2011年12月の売上はデータ期間の途中であるため、前月比で減少しているように見えますが、これはデータ取得期間の影響である可能性があります。

**洞察**: ピーク月の売上をさらに伸ばす戦略、低迷月の売上を底上げする施策、そして年末商戦に向けた早期準備が収益最大化に繋がります。

### 3.2. 高収益商品/カテゴリの特定

**分析目的**: 売上を牽引する商品を特定し、リソース配分を最適化する。

**結果**: 
*   売上トップ10の商品には、`DOTCOM POSTAGE`や`POSTAGE`といった**配送関連費**が2つランクインしています。これは売上計上方法の特性を示しています。
*   物理商品としては、`REGENCY CAKESTAND 3 TIER`、`PAPER CRAFT , LITTLE BIRDIE`、`WHITE HANGING HEART T-LIGHT HOLDER`などが高い売上を上げています。
*   `Manual`もトップ10に入っており、何らかのマニュアル販売も高収益であることがわかります。

**洞察**: 高収益の物理商品は、適切な在庫管理とプロモーションにより、さらに売上を伸ばす余地があります。また、配送関連費が高いことは、配送戦略の最適化がコスト削減に繋がり、間接的に利益率を向上させる可能性を示唆します。

### 3.3. 優良顧客の特定 (RFM簡易版)

**分析目的**: 顧客の購買行動を分析し、優良顧客を特定することで、パーソナライズされたマーケティング戦略を立案する。

**結果**: 
*   Recency（最終購入日からの日数）、Frequency（購入回数）、Monetary（合計購入額）の分布を確認しました。
*   大多数の顧客はRecencyが短く（最近購入しており）、FrequencyとMonetaryは少ない傾向にあります。これは、「パレートの法則」が示唆するように、**少数の優良顧客が売上の大部分を占めている**可能性を示しています。
*   Monetaryのヒストグラムでは、一部の高額購入者が存在するロングテール分布が見られます。

**洞察**: 優良顧客（Recencyが短く、Frequency/Monetaryが高い顧客）を特定し、彼らのロイヤルティを高める施策は、効率的な収益確保に不可欠です。また、Recencyが長い顧客やFrequency/Monetaryが低い顧客への再活性化も重要です。

## 4. アクションプラン：現場が明日から実行できる具体的な提案

これまでの分析結果に基づき、売上および利益を向上させるための具体的なアクションプランを提案します。

### 4.1. 売上トレンド分析からのアクション

*   **ピーク月のプロモーション強化**: 11月の高売上トレンドを最大限に活用するため、早めにマーケティング予算を集中させ、ターゲット広告、特別セール、限定商品の投入などを実施します。
*   **低迷月のキャンペーン**: 2月と4月の売上落ち込みに対しては、季節のイベント（例: バレンタイン、新生活、母の日など）に合わせたテーマ性のあるキャンペーンを展開し、顧客の購買意欲を刺激します。
*   **年末商戦の早期準備**: 前年度の12月データを詳細に分析し、在庫計画、人員配置、プロモーション内容を早期に決定・実行することで、機会損失を防ぎ、年末商戦を成功に導きます。

### 4.2. 高収益商品分析からのアクション

*   **高収益商品の在庫確保とバンドル販売**: `REGENCY CAKESTAND 3 TIER`などの高収益物理商品は、常に十分な在庫を確保し、品切れによる販売機会損失を回避します。また、これらの商品と関連性の高い商品を特定し、セット販売やクロスセルを強化することで、客単価向上を目指します。
*   **配送コストの最適化**: `DOTCOM POSTAGE`や`POSTAGE`が高い売上を占めることから、配送業者との料金交渉、一定金額以上購入時の送料無料サービス導入、複数商品の同梱奨励などを通じて、配送コストの削減と顧客満足度向上を図ります。

### 4.3. 優良顧客分析（RFMの簡易版）からのアクション

*   **ロイヤルティプログラムの導入**: 高頻度・高額購入者（優良顧客）向けに、VIP会員制度、限定イベントへの招待、バースデーギフト、専用サポートなどを提供し、彼らのブランドへの愛着と継続的な購買を促進します。
*   **離反顧客の再活性化**: 長期間購入のない顧客に対しては、過去の購入履歴に基づいたパーソナライズされたメールキャンペーン（例: 「お久しぶりですねクーポン」、新商品の紹介）や、特別な割引を提供し、再購入を促します。
*   **新規顧客の育成**: Recencyが短い新規顧客に対しては、購入後のフォローアップを強化し、継続的な関係構築のためのエンゲージメント施策（例: ニュースレター登録特典、次回の購入に使える割引券）を提供します。

## 5. 利益を上げるための新たな分析提案（異なる着眼点から）

*   **商品ペアリング/アソシエーション分析**: どの商品が一緒に購入されるかを分析し、**客単価向上**のためのバンドル販売やレコメンデーションを最適化します。 (KGI: 客単価の向上、クロスセル率の改善)
*   **地域別売上・顧客特性分析**: 国や地域ごとの売上傾向、人気の高い商品、顧客層の特性を深掘りし、**地域別マーケティング戦略**の最適化を目指します。 (KGI: 地域ごとの売上最大化、市場シェア拡大)
*   **在庫最適化とコスト削減分析**: 商品ごとの在庫回転率、リードタイム、季節変動を分析し、**在庫コスト削減**と品切れによる販売機会損失の最小化を図ります。 (KGI: 在庫コストの削減、粗利益率の改善)
*   **顧客生涯価値（LTV）に基づくセグメンテーション**: 顧客のLTVを予測し、高LTV顧客には維持・育成のための施策を、低LTV顧客には引き上げのための施策を効率的に実施し、**顧客生涯価値の最大化**を目指します。 (KGI: LTVの向上、顧客維持率の改善)
*   **プロモーション効果測定と最適化**: 過去のプロモーションが売上、利益、顧客行動に与えた影響を定量的に分析し、**費用対効果の高いプロモーション戦略**を特定します。 (KGI: プロモーションROIの最大化、広告費用対効果の改善)

## 6. ビジネスインパクトと最終評価指標 (KGI)

提案されたアクションプランと新たな分析提案は、以下のKGI（Key Goal Indicator）に集約され、収益の最大化に貢献します。

*   **収益の最大化のための分析**: 
    *   月次売上成長率の向上
    *   客単価の向上
    *   クロスセル/アップセル率の増加
    *   新規顧客獲得コストの最適化
    *   顧客維持率の向上
    *   顧客生涯価値（LTV）の向上

*   **コストの最適化のための分析**:
    *   在庫コストの削減
    *   配送コストの削減
    *   マーケティング/プロモーションROIの改善

### 機会費用と投資対効果（ROI）の仮説

各提案は、具体的な売上インパクトとコスト（工数）を考慮した投資対効果の仮説に基づいています。

*   **ピーク月のプロモーション強化**: 
    *   **売上インパクト**: ピーク月の売上を前年比で5-10%増加。
    *   **コスト**: 追加のマーケティング費用、プロモーション企画工数。
    *   **ROI**: 集中投資による高い売上リターンを期待。
*   **優良顧客向けロイヤルティプログラム**: 
    *   **売上インパクト**: 優良顧客の購入頻度を10%、購入単価を5%向上。
    *   **コスト**: システム開発費用、特典費用、運用工数。
    *   **ROI**: 新規顧客獲得より低コストで、安定した収益源を確保。
*   **在庫最適化**: 
    *   **売上インパクト**: 品切れによる販売機会損失を年間2%削減。
    *   **コスト**: 在庫管理システム改善費用、分析工数。
    *   **ROI**: 無駄な保管コスト削減と販売機会損失の防止による利益率改善。

これらの仮説は、提案実行後のA/Bテストや効果測定を通じて検証され、継続的な改善サイクルに組み込まれます。
